# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# 原始商品價目表
products_info = {"蘋果手機": 35000, "藍牙耳機": 3000, "行動電源": 1200, "手機殼": 650}

discounted_products = {name: int(price * 0.9) for name, price in products_info.items() if price > 1000}

# 結果: {'蘋果手機': 31500, '藍牙耳機': 2700, '行動電源': 1080}
```

## 使用的 Python 技術

- dict comprehension(字典綜合表達式)
- `.items()` 同時取出 key 與 value
- 在綜合表達式中對 value 做運算(`price * 0.9`)
- `int()` 型別轉換
- `if` 條件篩選(`if price > 1000`)

## 學習目標

學生可以學會:

- 用一行字典綜合表達式產生新的字典
- 在表達式中對原字典的值做數學運算後重新對應
- 用 `if` 條件決定哪些項目要被放進新字典

# 本次練習題目(全新題目,與課堂範例不同)

> 說明:以下是針對相同教學重點設計的「新題目」,情境與資料和課堂範例不同,但用到的 Python 語法一致。

## 題目敘述

某補習班有一份學生的考試成績,記錄在字典裡(key 是學生姓名,value 是分數)。老師想替「及格(60 分以上)」的學生做加分,規則是:及格學生的分數一律乘以 1.05(加 5% 鼓勵分),並四捨五入取整數(用 `int()` 無條件捨去即可)。

請用一行字典綜合表達式,從原始成績中篩選出及格的學生,並產生一份新的「加分後成績字典」。不及格的學生不放進新字典。

範例資料:

```python
scores = {"小明": 85, "小華": 58, "小美": 72, "阿杰": 49, "小芳": 90}
```

預期結果(參考):`{'小明': 89, '小美': 75, '小芳': 94}`

## 與課堂範例的對應

- 沿用的語法/概念:dict comprehension、`.items()`、表達式中對 value 運算、`int()` 轉換、`if` 篩選條件。
- 換掉的部分:情境從「商品打折」換成「成績加分」;運算從打 9 折(`* 0.9`)換成加 5%(`* 1.05`);篩選條件從「價格 > 1000」換成「分數 >= 60」。

# 延伸挑戰題

## 題目

延續上面的成績情境,這次老師想做「分級標記」而不是加分:用一行字典綜合表達式,把每位學生(全部都要,不篩選)轉換成新字典,key 維持姓名,value 改成等第文字 —— 90 分以上是 `"優"`、60 到 89 分是 `"及格"`、60 分以下是 `"待加強"`。

預期結果(參考):`{'小明': '及格', '小華': '待加強', '小美': '及格', '阿杰': '待加強', '小芳': '優'}`

## 提示

可以思考:

- 在綜合表達式的「value 位置」能不能放一個三元運算式(`A if 條件 else B`)?
- 要分三級時,三元運算式可以怎麼巢狀(`"優" if score>=90 else "及格" if score>=60 else "待加強"`)?
- 這題不需要 `if` 篩選,因為每位學生都要保留,差別在 value 怎麼決定。
