# 範例分析

## 原始程式

```python
#!usr/bin/python3
'''
讓使用者輸入梯形的上底、下底及高，程式會計算梯形的面積(上底加下底乘以高除以2)
'''
top = float(input('請輸入梯形的上底(公分):'))
bottom = float(input('請輸入梯形的下底(公分):'))
height = float(input('請輸入梯形的高(公分):'))
area = (top + bottom) * height / 2
print('梯形的面積:', area, '平方公分')
```

## 使用的 Python 技術

- `input()`：讀取使用者輸入
- `float()`：將文字轉換成浮點數
- 變數（variable）：儲存上底、下底、高、面積
- 算術運算子：`+`、`*`、`/`
- 運算優先順序與括號 `( )`
- `print()`：輸出結果

## 學習目標

學生可以學會：

- 用 `float()` 把輸入的文字轉成可計算的數字
- 把數學公式翻譯成 Python 算術運算式
- 用括號控制運算順序，正確計算結果並輸出

## 提示

可以思考：

- 三角形面積公式和梯形面積公式在寫法上有什麼相同、不同？
