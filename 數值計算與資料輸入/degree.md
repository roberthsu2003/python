# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
#!usr/bin/python3
'''
讓使用者輸入直角三角形的對邊與斜邊,
利用 sin(x) = 對邊 / 斜邊 計算夾角,
再把弧度(radian)轉換成角度(degree)印出。
'''

import math

opposite = float(input('請輸入直角三角形的對邊長度:'))
hypotenuse = float(input('請輸入直角三角形的斜邊長度:'))

radian = math.asin(opposite / hypotenuse)
degree = math.degrees(radian)

print('夾角的弧度:', radian, '弧度')
print('夾角的角度:', degree, '度')
```

## 使用的 Python 技術

- `import` 匯入內建模組(math)
- `math.asin()`、`math.degrees()` 呼叫模組函式
- `input()` 讀取使用者輸入
- `float()` 文字轉浮點數
- 變數(variable)
- `print()` 輸出結果

## 學習目標

學生可以學會:

- 用 `import` 把內建模組引入程式中
- 透過「模組名稱.函式名稱()」的方式呼叫模組裡的函式
- 結合 `input()`、型別轉換與模組函式完成一個數學運算任務

## 題目敘述

公園裡有一棵大樹,你想知道它有多高,但不方便直接爬上去量。你站在距離樹根某個水平距離的位置,抬頭看樹頂,用測角器量出「仰角」(從水平線往上看的角度)。

請讓使用者輸入「與樹的水平距離(公尺)」和「看樹頂的仰角(角度)」,利用 `tan(角度) = 樹高 / 水平距離`,計算並印出這棵樹的高度。

提示:`math.tan()` 接受的參數是「弧度」,所以要先把使用者輸入的角度用 `math.radians()` 轉成弧度,才能丟進 `math.tan()`。

## 與課堂範例的對應

- 沿用的語法/概念:`import math`、`input()`、`float()`、呼叫 math 模組函式、`print()`
- 換掉的部分:情境從「直角三角形求夾角」換成「測量樹高」;函式從 `math.asin()`/`math.degrees()` 換成 `math.radians()`/`math.tan()`;這次是「角度轉弧度」(與課堂範例的「弧度轉角度」方向相反)
