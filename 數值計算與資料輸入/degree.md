# 範例分析

## 原始程式

```python
#!usr/bin/python3
'''
讓使用者輸入直角三角形的對邊與斜邊，
利用 sin(x) = 對邊 / 斜邊 計算夾角，
再把弧度(radian)轉換成角度(degree)印出。
'''

import math

opposite = float(input('請輸入直角三角形的對邊長度:'))
hypotenuse = float(input('請輸入直角三角形的斜邊長度:'))

radian = math.asin(opposite / hypotenuse)
degree = math.degrees(radian)

print('夾角的弧度:', radian, '弧度')
print('夾角的角度:', degree, '度')
```

## 使用的 Python 技術

- import math
- input()
- float()
- math.asin()(反正弦,由比值求弧度)
- math.degrees()(弧度轉角度)
- 除法運算子(/)
- variable(變數)

## 學習目標

學生可以學會:

- 用 `import math` 引入數學模組並使用其中的函式
- 理解 `sin(x) = 對邊 / 斜邊`,並用 `math.asin()` 反推角度的弧度
- 用 `math.degrees()` 把弧度轉成一般人習慣的角度
