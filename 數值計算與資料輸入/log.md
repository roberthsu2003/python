# 範例分析

## 範例情境說明

這個範例來自一個複利存款問題:

- 假設存款現在有 20000 元,每年有 1.1% 的利息,以複利計算
- n 年後的存款為 `20000 × 1.011ⁿ`
- 想知道「不另外存錢,何時會達到 22000 元」,可列式:`20000 × 1.011ⁿ = 22000`
- 兩邊同除以 20000 → `1.011ⁿ = 1.1`
- 兩邊取對數(log A = log B)→ `n × log(1.011) = log(1.1)`
- 解出 `n = log(1.1) / log(1.011) ≈ 8.71`(年)

我們用 Python 把這個推導過程算出來。

## 原始程式

```python
#!usr/bin/python3
'''
複利存款問題：
存款 20000 元，年利率 1.1%（複利），
不另外存錢，計算幾年後存款會達到 22000 元。

利用對數求解：
20000 × 1.011^n = 22000
1.011^n = 1.1
n × log(1.011) = log(1.1)
n = log(1.1) / log(1.011)
'''

import math

principal = float(input('請輸入目前存款金額:'))
rate = float(input('請輸入年利率(例如 1.1% 請輸入 0.011):'))
target = float(input('請輸入目標金額:'))

# 倍數 = 目標金額 / 目前存款
ratio = target / principal

# 利用對數求出年數 n
n = math.log(ratio) / math.log(1 + rate)

print('大約需要', n, '年才能達到目標金額')
```

## 使用的 Python 技術

- import math
- input()
- float()
- math.log()(對數)
- 除法與加法運算子(/、+)
- variable(變數)
