# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# 1~100 偶數的加總
# 1~100 奇數的加總
evenTotal = 0
oddTotal = 0

for i in range(1, 101):
    if i % 2 == 0:
        evenTotal += i
    else:
        oddTotal += i

print(f"1加到100偶數的加總為 {evenTotal}")
print(f"1加到100奇數的加總為 {oddTotal}") 
```

## 使用的 Python 技術

例如:

- 變數
- for
- range()
- if / else
- %
- +=
- print()
- f-string

## 學習目標

學生可以學會:

- 使用 `for...in` 搭配 `range()` 重複處理一段數字
- 在迴圈中使用 `if / else` 判斷不同條件
- 依照不同條件，將數字分別累加到不同變數中

### 完成後參考程式碼(教師用,勿發給學生)

```python
# 建立一個變數，用來記錄 3 的倍數總和

notMultipleOfThreeTotal = 0

# 輸出不是 3 的倍數總和

```
