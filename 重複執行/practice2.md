# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# =======================================================================
# Name        : forNest2.py
# 利用2層迴圈列印九九乘法表
# =======================================================================

for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i:<2d}*{j:2d} ={i*j:2d}", end='  ')
    print()
```

## 使用的 Python 技術

例如:

- print()
- for 迴圈
- range()
- 巢狀迴圈
- f-string 格式化輸出
- end 參數
- 乘法運算

## 學習目標

學生可以學會:

- 使用兩層 `for` 迴圈產生表格資料
- 理解外層迴圈與內層迴圈的執行順序
- 使用 `print(..., end='  ')` 控制同一列輸出
- 使用 `print()` 換行，完成表格式顯示
