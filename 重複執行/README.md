###  重複執行概念
- 重覆執行可以讓程式中某一區段流程反覆執行
- 重複執行可用 while 及 for 兩種方式設計

### while 迴圈的架構如下:
```python
設定控制迴圈變數的初始值
while 條件分析:
	條件成立下執行迴圈要執行的工作
	調整控制迴圈變數的內容
```

### while 迴圈
- 當 while 迴圈的條件分析成立時，才會進入到迴圈。
- 若 while 迴圈的條件分析不成立，則跳出迴圈。
- 若 while 迴圈內條件分析一開始不成立，則迴圈一次都不執行。

###  迴圈中斷
- break 與 continue 都是迴圈中斷語法
	- break 代表迴圈中斷後跳出迴圈。
	- continue 代表迴圈中斷後繼續執行迴圈。


```python
#迴圈中斷:請留意輸出結果
#break.py

i = ['a', 'b', 'c', 'd'] 
for j in i:
    if j == 'c': 
        break
    print(j)

```


```python
#迴圈中斷:請留意輸出結果
#continue.py

i = ['a', 'b', 'c', 'd'] 
for j in i:
    if j == 'c': 
        continue
    print(j)
```
