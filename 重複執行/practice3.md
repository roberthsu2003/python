# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# 使用時機，不明確知道要執行幾次迴圈
# 配合條件式 if，並使用 break 停止迴圈

while(True):
    stuff = input("請輸入小寫英文字[按q會離開]: ")
    if stuff == 'q':
        break
    print(stuff.capitalize())
print("程式結束")
```

## 使用的 Python 技術

例如:

- while True
- input()
- variable
- if
- break
- print()
- 字串方法 `.capitalize()`

## 學習目標

學生可以學會:

- 使用 `while True` 建立不確定執行次數的迴圈
- 使用 `if` 判斷使用者是否輸入離開指令
- 使用 `break` 中斷迴圈並結束程式流程

# 延伸挑戰題

## 題目

請修改程式,讓使用者輸入英文單字時,如果輸入的是空字串,也就是直接按 Enter,程式不要建立單字卡,而是印出 `請不要輸入空白` 後繼續要求下一次輸入。輸入 `stop` 時仍然結束程式。

## 提示

可以思考:

- 空字串可以用 `word == ""` 判斷
- 可以在 `while True` 裡面放入多個 `if`
- 哪一種情況要 `break`,哪一種情況只要印出提醒訊息
