# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
score = int(input("請輸入分數"))
if score >= 90:
    grade = '優'
elif score >= 80:
    grade = '甲'
elif score >= 70:
    grade = '乙'
elif score >= 60:
    grade = '丙'
else:
    grade = '丁'

print("成績等級為：", grade)
```

## 使用的 Python 技術

- input()
- int() 型別轉換
- variable(變數)
- if / elif / else 多分支判斷
- 比較運算子(>=)
- print()

## 學習目標

學生可以學會:

- 用 if / elif / else 把一個數值範圍切成多個區段
- 理解 elif 的判斷順序(由上往下,符合就停止)
- 依不同條件指定不同結果並輸出

## 題目敘述

便利商店推出「飲料杯數折扣」活動,請寫一支程式,讓使用者輸入一次購買的飲料杯數,程式依照下列規則判斷並輸出折扣方案:

- 20 杯(含)以上:打 7 折
- 10 杯(含)~ 19 杯:打 8 折
- 5 杯(含)~ 9 杯:打 9 折
- 1 杯(含)~ 4 杯:原價(不打折)
- 0 杯:輸出「沒有購買任何飲料」

請輸入飲料杯數,輸出對應的折扣方案文字。

## 與課堂範例的對應

### 完成後參考程式碼(教師用,勿發給學生)

elif cups >= 1:

else:
    discount = '沒有購買任何飲料'

# 延伸挑戰題

## 題目

把程式升級成「自動算出實付金額」:讓使用者除了輸入杯數,再輸入每杯單價,程式依折扣規則算出折扣率,並輸出原價總額與折扣後的實付金額(四捨五入到整數元)。

## 提示

可以思考:

- 折扣率可用數字表示(例如 7 折就是 0.7、原價就是 1.0),用 if / elif / else 先決定折扣率變數
- 實付金額 = 杯數 × 單價 × 折扣率
- 四捨五入可查 round() 的用法
