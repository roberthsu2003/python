# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
chinese = int(input("請輸入國文分數"))
math = int(input("請輸入數學分數"))
#使用邏輯運算子
bonus = 0

if chinese == 100 and math == 100:
    bonus = 1000
elif chinese == 100 or math == 100:
    bonus = 500
print(f"獎金:{bonus}")
```

## 使用的 Python 技術

例如:

- input()
- int()
- print()
- variable
- if
- elif
- and
- or
- 比較運算子 `==`

## 學習目標

學生可以學會:

- 使用 `and` 判斷兩個條件是否同時成立
- 使用 `or` 判斷其中一個條件成立即可
- 搭配 `if`、`elif` 設計不同條件下的輸出結果

## 題目敘述

請設計一個「遊樂園入場優惠判斷程式」。

讓使用者輸入年齡與身高。如果年齡小於等於 12 歲，而且身高小於等於 140 公分，票價為 100 元。
