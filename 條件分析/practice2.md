# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
#學生總分為300
#有些學生可以加分5%
#如果加分超過300,就以300分為準
try:
    score = int(input("請輸入學生分數(最高300分):"))
    if score <= 300:
        is_add = input("學生是否符合加分條件?(y,n)")

        if(is_add == "y"):
            score *= 1.05
            if score > 300:
                score = 300

        print(f"學生分數是{round(score)}")
    else:
        print("學生分數不可以大於300")
except:
    print("格式錯誤")
```

## 使用的 Python 技術

- input()
- int() 型別轉換
- print() 與 f-string
- variable(變數運算 `*=`)
- if / else 判斷
- try...except 例外處理
- round()

## 學習目標

學生可以學會:

- 用 try...except 攔截使用者輸入錯誤,避免程式中斷
- 在 try 區塊中結合 if 判斷做資料合理性檢查
- 對輸入做型別轉換與數值運算

# 延伸挑戰題

## 題目

擴充程式,讓使用者可以**連續輸入多筆訂單**:每次輸入一筆杯數,輸入 `q` 結束。結束後印出所有訂單的**總杯數**與**總金額**。每筆訂單同樣套用「最多 20 杯」「滿 10 杯打 9 折」「輸入錯誤要攔截」的規則,且某一筆輸入錯誤時不可讓整個程式中斷,要能繼續輸入下一筆。

## 提示

可以思考:

- 用 while 迴圈不斷接收輸入,輸入 `q` 時 break
- 把 try...except 放在迴圈內,讓單筆錯誤不影響後續輸入
- 用兩個累加變數分別記錄總杯數與總金額
