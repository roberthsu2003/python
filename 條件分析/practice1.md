# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
#學生總分為300
#有些學生可以加分5%
#如果加分超過300,就以300分為準

scores = int(input("請輸入學生分數(最高300分):"))
is_add = input("學生是否符合加分條件?(y,n)")

if(is_add == "y"): #單項選擇
    scores *= 1.05
    if scores > 300:
        scores = 300
        
print(round(scores))
```

## 使用的 Python 技術

- input()
- int() 型別轉換
- print()
- variable(變數)
- if(條件判斷)
- 巢狀 if(if 裡面再放 if)
- 比較運算子(==、>)
- 算術運算(*= 、round())

## 學習目標

學生可以學會:

- 用 if 判斷使用者輸入,決定是否執行某段程式
- 在 if 裡面再放一個 if(巢狀判斷),處理「上限/下限」這類額外條件
- 用 round() 處理小數結果

## 題目敘述

百貨公司在做週年慶折扣。請寫一支程式,先請使用者輸入「消費金額」,再詢問使用者「是否為會員?(y,n)」。

如果是會員(輸入 y),全部金額打 9 折(乘以 0.9)。但公司規定:折扣後的最低應付金額不得低於 100 元,如果打折後低於 100,就以 100 元計算。

最後輸出使用者實際要付的金額(四捨五入到整數)。

## 與課堂範例的對應

- 沿用的語法/概念:input()、int()、if 單項選擇、巢狀 if 處理上下限、算術運算、round()、print()
- 換掉的部分:情境從「學生加分」換成「會員打折」;原範例是「加分後不能超過上限 300」,新題目是「打折後不能低於下限 100」——一個是檢查上限,一個是檢查下限,讓學生反向思考巢狀 if 的條件。

## 題目

擴充上面的程式,加入「會員等級」判斷。詢問使用者是否為會員後,如果是會員,再請使用者輸入會員等級(輸入 1 或 2):

- 等級 1(一般會員):打 9 折
- 等級 2(VIP 會員):打 8 折

折扣後一樣不得低於 100 元,最後印出實付金額。

## 提示

可以思考:

- 確認是會員之後,要再用一個 if 去判斷等級,這會變成第幾層的巢狀 if?
- 不同等級對應不同折扣數字(0.9 或 0.8),要怎麼用 if / else 分開處理?
- 「不得低於 100」的那段判斷,應該放在折扣算完之後,對所有等級都適用
