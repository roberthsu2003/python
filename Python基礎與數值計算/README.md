本單元介紹 Python 的基礎語法、變數概念、互動式開發環境、基本數值運算以及資料的輸入與輸出。

## 1. 開發環境與基礎工具

### 1.1 互動式環境 IPython / Jupyter
在 Python 開發中，除了傳統的 Python Shell 外，我們常使用功能更強大的 **IPython** 或 **Jupyter Notebook** 作為互動式開發環境。它們提供了許多便利的功能：

* **自動輸出**：不需使用 `print()`，直接輸入變數名稱或運算式即可顯示內容。
* **Tab 鍵自動完成**：輸入模組、函式或變數的前幾個字，按下 `<Tab>` 即可獲得語法提示與補全。
* **`?` 檢查機制**：在變數或函式名稱後加上 `?`，可快速查看其型態、長度與說明文件（Docstrings）。
* **魔法指令 (Magic Commands)**：
  - `%run`：在 IPython 中直接執行外部 Python 腳本（例如 `%run script.py`）。
  - `%load`：在 Jupyter Notebook 中將外部檔案的內容載入至當前儲存格中。

```python
# IPython 操作範例
In [1]: import numpy as np
In [2]: data = {i: np.random.randn() for i in range(3)}
In [3]: data?  # 快速檢查
Type:        dict
String form: {0: 0.6521, 1: -0.1235, 2: 1.0921}
Length:      3
```

### 1.2 註解 (Comments) 的使用
註解是程式的說明文字，不會被 Python 解譯器執行。撰寫良好的註解是良好程式習慣的開始：

* **單行註解**：以井字號 `#` 開頭，該行在其後的內容皆為註解。
* **多行註解**：Python 原生不支援特別的多行註解語法，本講義採用以下常用做法：
  1. 在每行開頭使用 `#`。
  2. 使用**未指派給變數的多行字串**（以三個單引號 `'''` 或雙引號 `"""` 包裹），當解譯器讀到未被指派的多行字串時會將其忽略，因而可作為多行註解使用。

```python
# 這是單行註解
seconds_per_day = 86400  # 24 * 60 * 60

"""
這是一個多行註解的範例。
可以用於詳細解釋函式或類別的設計邏輯。
"""
```

## 2. 變數與資料型別

### 2.1 變數的宣告與記憶體參照
![](./images/variable_concept.png)

在 Python 中，變數的操作有以下幾個核心特性：
1. **不需事先宣告**：與 C++、Java 等語言不同，Python 變數不需要指定資料型態，即可直接使用。
2. **動態綁定與記憶體參照**：變數本質上是一個**指標（參照）**，指向記憶體中的某個物件。使用指派運算子 `=` 代表「將右側物件的參照，交給左側的變數名稱」。
3. **記憶體釋放**：如果不再需要某個變數，可以使用 `del` 關鍵字將其刪除，以節省記憶體。
4. **觀測記憶體位址**：可使用內建函式 `id(變數)` 查詢變數目前所指向的記憶體位址；使用 `type(變數)` 查詢其目前的資料型態。

```python
a = 100
b = a  # b 與 a 指向相同的記憶體位址 (id 一致)

a = 200  # a 轉而指向新建立的 200 物件，此時 b 仍指向 100
print(id(a))  # 與前一次不同
```

### 2.2 變數命名規則
* 第一個字元**不能是數字**。
* 只能使用大小寫字母、數字與下底線 `_`。
* 不可與 Python 的內建保留字（如 `if`, `for`, `def`, `class`, `import` 等）同名。
* 大小寫視為不同的變數（例如 `years` 與 `Years` 是不同的變數）。
* 不能包含空白或特殊符號。
* *註：Python 3 開始支援使用中文命名變數，但實務上為保持相容性，仍建議以英文命名。*

#### 📝 範例對比：
* **合法命名**：`a`, `a1`, `a_b_c`, `_abc`, `巨匠`
* **不合法命名**：`1a` (數字開頭), `if` (保留字), `pcschool&python` (包含特殊字元), `Pcschool python` (包含空白)

### 2.3 內建資料型別與型別轉換

Python 最基礎的內建資料型別包括：
* **整數 (`int`)**：包含正負整數。Python 的整數長度沒有限制，能自動處理極大數值的運算。
  - **進位制表示**：`0b` (二進位), `0o` (八進位), `0x` (十六進位)。
* **浮點數 (`float`)**：即包含小數點的數值。
* **布林值 (`bool`)**：僅有 `True` 與 `False` 兩種狀態（在數值計算中，`True` 相當於 `1`，`False` 相當於 `0`）。
* **字串 (`str`)**：文字資料，可用單引號 `'` 或雙引號 `"` 包裹。
  - **多行文字**：可用三個單/雙引號包裹，或在行尾使用反斜線 `\` 進行折行連結。
  - **跳脫字元**：以反斜線 `\` 引用特殊字元，例如 `\n` (換行)、`\t` (定位鍵)、`\'` (單引號)、`\"` (雙引號)、`\\` (反斜線)。

```python
# 進位制與大整數
print(0b10)     # 輸出 2
googol = 10**100  # 自動處理大數

# 類型強轉函式
int("13") + int("25")       # 轉為整數，輸出 38
float("13.5") + float("2.2") # 轉為浮點數，輸出 15.7
str(98.6)                   # 轉為字串，輸出 '98.6'
```

#### ⚠️ 浮點數底層誤差與 `Decimal` 精度運算
由於計算機底層以二進位儲存浮點數，部分十進位小數無法被精確表達，會產生微小誤差（例如 `1.1 + 2.2` 得到的結果是 `3.3000000000000003`）。
對於金融、貨幣等高精度需求的計算，應使用內建的 `decimal` 模組：

```python
from decimal import Decimal

# 注意：必須傳入字串格式，否則仍會帶入浮點數本身的二進位誤差
result = Decimal('1.1') + Decimal('2.2')
print(result)  # 輸出 3.3
```

## 3. 資料輸入與數值運算

### 3.1 資料輸入 `input()`
* 使用 `input("提示字串")` 接受使用者的鍵盤輸入。
* **⚠️ 重要觀念**：`input()` 讀入的資料型別**預設一律為字串 (`str`)**。
* 若要進行數值計算，必須明確以 `int()` 或 `float()` 將輸入結果進行轉型。

```python
# input_test.py
age_str = input("請輸入年齡: ")
print("型態為:", type(age_str))        # 輸出 <class 'str'>

age = int(input("請輸入整數年齡: "))  # 讀入後直接轉型為 int
print("明年年齡:", age + 1)
```

#### 💡 實戰範例 1：兩數相加
計算使用者輸入的兩個任意數，並顯示兩數相加的總和：
```python
n = float(input('請輸入第一個數值: '))
m = float(input('請輸入第二個數值: '))
print('兩個數的和是: ', n + m)
```

#### 💡 實戰範例 2：平方與立方計算
使用者輸入一個任意數，顯示此數的平方值及立方值：
```python
num = float(input('請輸入任意數: '))
result = num ** 2
print('此數的平方是: ', result)
result = num ** 3
print('此數的立方是: ', result)
```

### 3.2 數學運算子與優先順序
Python 提供了豐富的數學運算子：

| 運算子 | 描述 | 範例 | 結果 |
| :---: | :--- | :--- | :--- |
| `+` | 加法 | `5 + 8` | `13` |
| `-` | 減法 | `90 - 10` | `80` |
| `*` | 乘法 | `4 * 7` | `28` |
| `/` | 浮點數除法 | `7 / 2` | `3.5` |
| `//` | 整除（只取商數） | `7 // 2` | `3` |
| `%` | 取餘數 | `7 % 3` | `1` |
| `**` | 次方（冪運算） | `3 ** 4` | `81` |

* **除數限制**：除數（分母）不可為零，否則會丟出 `ZeroDivisionError`。
* **`divmod()` 函式**：能同時取得商數與餘數，回傳一個元組 `(商, 餘數)`。例如 `divmod(9, 5)` 會得到 `(1, 4)`。
* **運算優先順序**：
  1. 括號 `()`
  2. 次方 `**`
  3. 正負號（例如 `-x`）
  4. 乘、除、整除、取餘 `*`, `/`, `//`, `%`
  5. 加、減 `+`, `-`
  6. 賦值與複合賦值 `=`

#### 💡 實戰範例 1：商與餘數計算
此範例示範如何以整除 `//` 與取餘數 `%` 計算兩數相除 the 商與餘數：
```python
n = int(input('請輸入被除數(整數): '))
m = int(input('請輸入除數(整數, 不可以是 0): '))
print('商:', n // m, '餘數:', n % m)
```

#### 💡 實戰範例 2：梯形面積計算
讓使用者輸入梯形的上底、下底與高，並計算其面積（$\text{面積} = (\text{上底} + \text{下底}) \times \text{高} / 2$）：
```python
top = float(input('請輸入梯形的上底(公分): '))
bottom = float(input('請輸入梯形的下底(公分): '))
height = float(input('請輸入梯形的高(公分): '))
area = (top + bottom) * height / 2
print('梯形的面積: ', area, ' 平方公分')
```

#### 💡 實戰範例 3：圓柱體體積計算
使用者輸入圓柱體的半徑及高，計算圓柱體的體積（$\text{體積} = \pi \times \text{半徑}^2 \times \text{高}$，其中 $\pi \approx 3.14159$）：
```python
PI = 3.14159
radius = float(input('請輸入圓柱體的半徑(公分): '))
height = float(input('請輸入圓柱體的高(公分): '))
area = radius ** 2 * PI * height
print('圓柱體的體積: ', area, ' 立方公分')
```

#### 💡 實戰範例 4：直角三角形角度計算 (使用 `math` 模組)
輸入直角三角形的對邊及斜邊長度，計算其夾角角度（公式：$\sin(x) = \text{對邊} / \text{斜邊}$，使用 `math` 模組轉換角度）：
```python
import math
opposite = float(input('請輸入直角三角形的對邊長度: '))
hypotenuse = float(input('請輸入直角三角形的斜邊長度: '))
radian = math.asin(opposite / hypotenuse)
degree = math.degrees(radian)
print('夾角的弧度: ', radian, ' 弧度')
print('夾角的角度: ', degree, ' 度')
```

#### 💡 實戰範例 5：對數利息運算 (使用 `math.log`)
計算存款在複利之下達到目標金額所需要的時間，並利用 `math.log()` 進行求解：
```python
import math
principal = float(input('請輸入目前存款金額: '))
rate = float(input('請輸入年利率(例如 1.1% 請輸入 0.011): '))
target = float(input('請輸入目標金額: '))

ratio = target / principal
n = math.log(ratio) / math.log(1 + rate)
print('大約需要 ', n, ' 年才能達到目標金額')
```

##### ✍️ 數學運算邏輯說明：
假設存款現在有 20,000 元，年利率為 1.1%，以複利計息，經過 $n$ 年後，存款總額會是：
$$20000 \times 1.011^n$$

若要計算存款何時能達到 22,000 元，可列出方程式：
$$20000 \times 1.011^n = 22000$$

兩邊同除以 20,000 以簡化方程式：
$$1.011^n = 1.1$$

根據等量公理，兩邊同時取對數 ($\log$)：
$$\log 1.011^n = \log 1.1$$
$$n \log 1.011 = \log 1.1$$

計算對數數值後：
$$0.00475n \approx 0.04139$$
$$n \approx 8.71\text{ (年)}$$

### 3.3 複合指定運算子
當「指派運算子 `=`」左右兩側出現同一個變數時，可以將運算子簡寫移至左側：
* `x = x + 2` ➡️ 可簡寫為 `x += 2`
* `y = y - 3` ➡️ 可簡寫為 `y -= 3`
* 相同的規則適用於其他運算子如 `*=`, `/=`, `%=`, `**=` 等。

#### 💡 實戰範例 1：手機餘額計算
```python
# money.py
money = 50000
cell = int(input("請輸入手機金額: "))
money -= cell  # 相當於 money = money - cell
print("剩餘款為: " + str(money))
```

#### 💡 實戰範例 2：三數相加（複合指定運算子）
讓使用者輸入三個任意數，並使用 `+=` 運算子顯示三數相加的總和：
```python
sum_val = 0
x = float(input('請輸入第一個數: '))
sum_val += x
x = float(input('請輸入第二個數: '))
sum_val += x
x = float(input('請輸入第三個數: '))
sum_val += x
print('三個數的總和為: ', sum_val)
```

## 4. 輸出資料格式化
將多個變數或不同型別的資料格式化為特定格式輸出，常見有三種做法：

### 4.1 傳統 `%` 格式化 (C語言風格)
語法：`"字串格式代碼" % (變數1, 變數2, ...)`
* `%d` 或 `%i`：十進位整數。
* `%f`：浮點數（可使用 `%.mf` 控制小數點後 `m`位數，亦可加入寬度控制，如 `%5.2f`）。
* `%s`：字串。
* `width` 對齊控制：正數為靠右對齊，負數為靠左對齊（如 `%-5s`）。

```python
# 浮點數格式化範例
b = 123.926
print("整數無條件捨去: %d" % b)       # 輸出: 123
print("保留2位小數(右對齊): %7.2f" % b) # 輸出:  123.93 (總寬度 7)
print("保留4位小數(左對齊): %-9.4f" % b) # 輸出: 123.9260  (總寬度 9)
```

### 4.2 新式 `.format()` 格式化
語法：`"佔位符".format(變數1, 變數2, ...)`
* **基本位置指定**：可以使用索引（`{0}`, `{1}`）或名稱（`{name}`）指定輸出位置。
* **數值格式化與對齊**：在 `{}` 內加入 `:` 及格式符。
  - `{:<8d}`：靠左對齊，寬度 8。
  - `{:>8d}`：靠右對齊，寬度 8。
  - `{:^8d}`：置中對齊，寬度 8。
  - `{:8.3f}`：總寬度 8，保留小數 3 位。

```python
# .format() 範例
print("{0} {1} {0}".format("A", "B"))           # 輸出: A B A
print("|{0:>8d}||{1:<8d}|".format(123, 456))    # 輸出: |     123||456     |
print("帶正負號: {0:+8.2f}".format(4.32))        # 輸出: 帶正負號:    +4.32
```

### 4.3 字串插補 f-string (Python 3.6+ 最推薦)
在字串前加上 `f` 開頭，直接在 `{}` 內寫入變數名稱或運算式，寫法最為直覺簡潔。

```python
name = "Python"
version = 3.9
print(f"Hello, {name}! Version is {version:.1f}")  # 輸出: Hello, Python! Version is 3.9
```
