# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
#!usr/bin/python3
'''
讓使用者輸入梯形的上底、下底及高，程式會計算梯形的面積(上底加下底乘以高除以2)
'''
top = float(input('請輸入梯形的上底(公分):'))
bottom = float(input('請輸入梯形的下底(公分):'))
height = float(input('請輸入梯形的高(公分):'))
area = (top + bottom) * height / 2
print('梯形的面積:', area, '平方公分')
```

## 使用的 Python 技術

- input()
- float()
- 變數(variable)
- 四則運算(+、*、/)
- print()

## 學習目標

學生可以學會:

- 用 input() 取得使用者輸入,並用 float() 轉成數字
- 把數學公式翻譯成 Python 運算式,注意運算順序與括號
- 用 print() 同時輸出文字與計算結果

## 題目敘述

請寫一支程式,計算**圓柱體的體積**。

讓使用者輸入圓柱底面的**半徑**與圓柱的**高**(單位都是公分),程式計算並印出圓柱體積。

圓柱體積公式為:**體積 = 圓周率 × 半徑 × 半徑 × 高**(圓周率取 3.14159)。

例如:半徑 5、高 10,體積約為 785.4 立方公分。

## 與課堂範例的對應

- 沿用的語法/概念:input() 取得輸入、float() 轉數字、變數儲存、四則運算、print() 輸出
- 換掉的部分:情境從「梯形面積」換成「圓柱體積」;輸入欄位改成半徑與高;公式改為含圓周率常數的連乘;輸出單位改為立方公分
