# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# 初始化待辦清單
todo_list = ["回覆客戶郵件", "準備下午會議簡報", "去銀行辦事"]

# 1. 新增今日臨時指派的任務 (使用 append)
todo_list.append("購買晚餐食材")
print("目前的待辦事項:", todo_list)

# 2. 順利完成第一項任務 (使用 pop(0) 移除並回傳)
completed_task = todo_list.pop(0)
print(f"【完成】: {completed_task}")
print("剩餘的待辦事項:", todo_list)
```

## 使用的 Python 技術

- list 初始化(方括號建立清單)
- `list.append()` 新增元素到清單尾端
- `list.pop(index)` 依索引移除並回傳該元素
- f-string 格式化輸出
- `print()` 多值輸出

## 學習目標

學生可以學會:

- 理解 list 會保留元素「加入的先後順序」
- 使用 `append()` 在清單尾端動態新增資料
- 使用 `pop(0)` 依照先後順序取出並移除最前面的資料
- 觀察清單在新增、刪除操作後內容如何變化
