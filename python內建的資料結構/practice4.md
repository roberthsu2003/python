# 課堂範例回顧(學生剛才已練習)

## 原始程式

```python
# 宣告愛麗絲與鮑伯的朋友集合
alice_friends = {"小明", "小華", "小強", "美美"}
bob_friends = {"小強", "美美", "大雄", "胖虎"}

# 1. 交集運算 (&)：尋找兩人的「共同好友」
mutual = alice_friends & bob_friends
print("共同好友:", mutual)  # 輸出: {'小強', '美美'}

# 2. 差集運算 (-)：尋找「愛麗絲有，但鮑伯沒有」的好友
unique_to_alice = alice_friends - bob_friends
print("僅愛麗絲擁有的好友:", unique_to_alice)  # 輸出: {'小明', '小華'}
```

## 使用的 Python 技術

- set 集合的建立(大括號 `{}`)
- `&` 交集運算子
- `-` 差集運算子
- `print()`

## 學習目標

學生可以學會:

- 理解 set 是「不允許重複元素」的資料結構,並知道為什麼這個特性適合用來存放「名單類」資料
- 使用 `&` 運算子,一行程式碼求出兩個集合的交集
- 使用 `-` 運算子,一行程式碼求出兩個集合的差集,並理解差集運算有方向性(A - B 不等於 B - A)

## 提示

可以思考:

- 想求出三個集合的共同交集,可以先用 `&` 算出前兩個集合的交集,再把結果拿去跟第三個集合做一次 `&`
- 「完全沒有參加另外兩社」可以連續使用兩次 `-`,例如 `coding_club - robot_club - science_club`
- 建議先把每一步的中間結果用 `print()` 印出來確認,再進行下一步運算,比較不容易出錯
